# Security Policy

## Supported Versions

Security mitigation is generally provided only for the latest minor version.

In rare cases where multiple Major versions are being supported, the latest minor version for each affected major version will recieve the security update.

In either case, a new minor version will be released including the bugfix.

## Reporting a Vulnerability

Please use the GitHub vulnerability reporting feature to ensure the fastest response to issues as well as preventing public disclosure before resolution. Since this is a volunteer driven project, please allow adequate time for a maintainer to respond to your vulnerability.
